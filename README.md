# Slash Printer
Converts the ASCII characters into 6-line text art using slash and underscore

ASCII 문자를 슬래시(/)와 밑줄(_)을 사용한 6줄의 텍스트 아트로 변환함

~~공부가 하기 싫었던 고3의 노가다 현장~~
